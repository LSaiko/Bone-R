Pipeline

1

Convert

FracAtlas COCO → YOLO boxes; MURA folders → abnormal/normal labels.

2

Train

YOLOv8 detector + MURA classifier; or fine-tune any torchvision backbone.

3

Ensemble

Classifier P(abnormal) re-weights detections — sensitivity-first fusion.

4

Serve

FastAPI /detect takes PNG/DICOM → boxes, severity, consult.
